For configurations:
    git config--global user.name
    git config--global user.email
For starting a project:
    git init [project name]
For Cloning:
    git clone [url]
For changing and status checking:
    git status
    git add .
    git commit -m"Change name"
For branching and Merging:
    git branch
    git branch [branch name] //new branch creating
    git checkout [branch name] //switching branch
    git merge [branch]
    git branch -d [branch name] //delete branch
Pushing and pulling:
    git push origin main
    git push origin master

Pull and rebase latest changes from remote branch 'main':
    git pull origin main --rebase
Fetch all branch from remote:
    git fetch origin
Add the remote repository:
    git remote add origin [url of repository]
    git remote -v //verify the remote repository
